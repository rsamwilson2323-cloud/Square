# ⬜ Crystal Square Ripple

An interactive HTML5 Canvas animation that creates glowing crystal-like square ripples with vibrant rainbow colors, neon lighting effects, and smooth expanding waves. Designed for both desktop and mobile devices, users can generate beautiful square ripple patterns through mouse and touch interactions.

## ✨ Features

* ⬜ Expanding crystal square ripples
* 🌈 Dynamic rainbow color transitions
* ✨ Neon glow lighting effects
* 🖱️ Mouse click and drag support
* 📱 Mobile touch interaction
* 🎨 Smooth fading ripple trails
* ⚡ Real-time Canvas rendering
* 🖥️ Fullscreen responsive design
* 🚀 Lightweight and dependency-free

## 🎮 Controls

### Desktop

* Click to create glowing square ripples.
* Hold and drag to continuously generate expanding square effects.

### Mobile

* Tap anywhere to generate crystal square waves.
* Touch and drag to create continuous glowing ripple animations.
